This documentation is a reference guide for guests using the audio and video technology in **Mackey Hall** at Westminster Presbyterian Church, Wooster, Ohio.

## What's Covered

<CardGrid>
  <Card title="Video" icon="laptop">
    Projector screen control, projector and rear TV power, and input source switching.
  </Card>
  <Card title="Audio" icon="information">
    Sound system startup, wall touchscreen controls, computer audio, Bluetooth pairing, and microphone setup.
  </Card>
  <Card title="Troubleshooting" icon="warning">
    Common issues and fixes — guides are being added. For immediate help, [email us](mailto:jack@wpcwooster.org).
  </Card>
  <Card title="Print / PDF" icon="document">
    A [print-ready version](/print/) of the full guide is available for download or offline printing.
  </Card>
</CardGrid>
