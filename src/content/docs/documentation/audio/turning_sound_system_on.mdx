title: Turning the Sound System On
description: How to verify and turn on the Mackey Hall sound system
---

import { Steps, Aside } from '@astrojs/starlight/components';

<Aside>If the system is already on, no changes are needed.</Aside>

## Check Rack Power

<Steps>
1. Go to the sound system rack.
2. Find the red `Power` switch on the rack.
3. If the switch is not lit red, turn it on.
</Steps>

![Rack Power Switch](/assets/images/audio/sound_system_on-off/power-new.png)

## Check Amplifier Power

<Steps>
1. Locate the amplifier in the rack (third unit from the bottom).
2. Press the amplifier power button at the bottom-right.
3. Confirm the button lights blue — this means the amplifier is on.
</Steps>

![Amplifier Power Button](/assets/images/audio/sound_system_on-off/power-new_amplifier.png)

## Sound System Ready

When the rack power is on and the amplifier button is blue, the sound system is ready for microphones, computer audio, and Bluetooth audio.
