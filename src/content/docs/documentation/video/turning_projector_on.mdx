If you need the reverse process, see [Turning Projector / Rear TV Off](/documentation/video/turning_projector_off/).

## Power On Projector

<Steps>
1. Go to the wall touchscreen in Mackey Hall.

2. If not on the Homepage or `Power` page, press the `Home` icon in the top-left corner.

3. Press `Power` to open power controls.

   ![Control Screen Homepage](/assets/images/video/control-screen_new/homepage.png)

4. Press `Projector On`.

   ![Power Menu Screen](/assets/images/video/control-screen_new/power_1.png)

5. Allow about 30 seconds for the projector to fully start.
</Steps>

<Aside type="caution">Do not turn off the projector immediately after turning it on. Allow it to fully start before shutting down.</Aside>

## Power On Rear TV

<Steps>
1. Go to the wall touchscreen in Mackey Hall.

2. If not on the Homepage or `Power` page, press the `Home` icon in the top-left corner.

3. Press `Power` to open power controls.

   ![Control Screen Homepage](/assets/images/video/control-screen_new/homepage.png)

4. Press `Rear TV`.

   ![Power Menu Screen](/assets/images/video/control-screen_new/power_1.png)

5. Allow a few seconds for the rear TV to fully power on.
</Steps>
